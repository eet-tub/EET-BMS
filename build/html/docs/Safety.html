Safety Instructions¶

  • configure the BMS settings exactly as the battery type allows

  • the maximum voltage and current must not exceed

  • the BMS in the current form should be only used as a research BMS

  • install the BMS and the batteries considering best practices for electrical installation in accordance to laws in your country

  • Use insulated tools only
